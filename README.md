# Scriptable Widgets

> A collection of simple and useful **Scriptable** widgets for iOS, built with JavaScript.

[Scriptable](https://scriptable.app) is an app to automate iOS using JavaScript and create custom widgets.

Each widget runs directly on your iPhone Home Screen, using JavaScript to create useful and personalized tools.

## Widgets

### Workout Tracker
A home-screen dot-grid showing a rolling 364-day workout history. Tap to open the companion logger, which supports plan, log, and review modes. Sessions are saved locally; the dot turns green once you've hit the day's threshold.
